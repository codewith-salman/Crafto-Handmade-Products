## 📖 About The Project

<img src="Crafto%20Handmade%20Products/assets/images/desktop.png" alt="Crafto Website Desktop View" width="100%" />

**Crafto** is a beautifully designed handmade products website UI, created to highlight unique crafts such as pottery, jewelry, accessories, home décor, and personalized handmade items.  
The design focuses on **warm color tones, minimal aesthetics, and visually rich product presentation**, making it ideal for artisans, craft sellers, and small handmade businesses.

The website layout emphasizes **product storytelling**, allowing users to explore categories, featured items, and craft inspirations in a smooth and immersive browsing flow.  
This project demonstrates your front-end skills in **responsive design, product card layouts, clean UI structure, and smooth animations**.

Ideal for:
- Handmade product stores  
- Small craft businesses  
- Art and DIY communities  
- E-commerce concept designs  
- Portfolio demonstration projects

## ✨ Features

- Fully responsive handmade product layout  
- Smooth hover animations and transitions  
- Category-wise product organization

- Lightweight and fast-loading structure
